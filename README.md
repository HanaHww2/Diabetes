# Diabetes
당뇨 환자를 위한 GL정보 제공, 혈당당 프로젝트



## GL이란
GL(Glycemic Load), '당부하지수'로 1회 분량을 기준으로 혈당 반응을 비교한 값으로, GL값이 낮을 수록 몸에 부담이 덜하다고 말할 수 있다.
